
DESCRIPTION:

Gabor.py is a program that illustrates what the Gabor Wavelet Transformation of
a simple image would resemble. It uses a sample input file (provided as 'wavelet.png')
to plot several components of the Gabor Wavelet Trasform side by side. The program,
along with the sample image, are provided in the 'src' folder in this directory

Example usage:
	
	python Gabor.py wavelet.png
